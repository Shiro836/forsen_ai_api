Quickstart
  • In your obs, choose scene where you want to put output window of AI.
  • Then navigate to Sources window, press "+" button and add new Browser source.
  • Put in URL field without quotes. Press "OK" button.
  • Go to page and add your first character.
  • Go to your chat and try to redeem your newly created reward. You should hear sound and see text.
  • Don't forget that you can skip undesired user requests in . And you can give access to your control panel in the settings below.
Images guide
  • AI can react to images in prompt. Images are not shown in obs overlay automatically and need to be approven in control panel to be shown.
  • Users need to upload images in . Copy the generated code that looks like <img:sdfns>
  • Use it in prompt preferably in the beginning of prompt like - <img:sdfns> opinion on thing
  • Images need to be manually approved in control panel. It's not very easy while streaming, so you can put people you trust to approve images in setting below.

How to set keybinds to hide obs overlay very fast
  • Open obs settings
  • Go to "Hotkeys" tab
  • Find "Hide {source name}" and "Show {source name}" hotkeys and set them to your desired keybinds
Setup global hotkeys to skip and show images
  • Download the OBS Lua script obs_script.lua and add it in OBS: Tools → Scripts → “+” → select the file.
  • In the script properties:
    • Select your BAJ AI Browser Source in the Browser Source dropdown.
    • Paste your Token from the .
  • Assign two global hotkeys in OBS Settings → Hotkeys:
    • BAJ_AI_SKIP — skips the current request.
    • BAJ_AI_SHOW_IMAGE — approves and shows prompt images for the current request.
  • Note that due to network latency, show image hotkey might show image of next message. You need to be careful with it.
